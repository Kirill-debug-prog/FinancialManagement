# FinancialManagement
Веб-приложение, которое помогает пользователям управлять личными и семейными финансами: вести учёт доходов и расходов, контролировать долги и кредиты, анализировать бюджет и планировать финансовые цели. Приложение доступно с любых устройств и не требует специальных знаний в бухгалтерии.
